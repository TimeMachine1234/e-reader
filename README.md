# PageTurn

An Android e-reader app supporting EPUB, PDF, TXT, and CBZ/CBR formats.

## Download

**[Download PageTurn v1.0.0](https://github.com/TimeMachine1234/e-reader/releases/tag/v1.0.0)**

Open the link on your Android device, tap `app-release.apk`, then tap the downloaded file to install.
> Requires Android 8.0 (Oreo) or later. You may need to allow installation from unknown sources when prompted.

## Features

- EPUB, PDF, TXT, CBZ/CBR format support
- Paginate and scroll reading modes
- Customizable fonts (Georgia, Palatino, Lato, Merriweather, OpenDyslexic, EB Garamond)
- Themes: Light, Dark, Sepia, E-Ink, E-Ink Warm
- Page curl animation
- Two-page landscape mode
- Highlights and bookmarks
- Volume button page turning
- Reading progress saved across sessions
- Blink reminder for eye comfort

## Building from Source

Requirements: Android Studio, JDK 17, Android SDK 35

```bash
git clone https://github.com/TimeMachine1234/e-reader.git
cd e-reader
./gradlew assembleDebug
```
